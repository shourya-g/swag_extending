# SWAG Modern Deep Learning Project

This project reproduces and extends SWAG (Stochastic Weight Averaging Gaussian) to evaluate its relevance in modern deep learning.

## Current Goals
- Reproduce SWAG on CIFAR-10 with ResNet-18
- Compare SGD, SWA, and SWAG
- Evaluate uncertainty using Accuracy, NLL, and ECE
- Extend SWAG with modern optimizers and architectures

## Project Structure
- `configs/` experiment configs
- `data/` datasets
- `notebooks/` analysis and visualizations
- `outputs/` checkpoints, plots, metrics
- `src/` source code
